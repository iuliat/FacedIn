Javascript Edge Detection

Javascript and HTML5 Canvas is used to detect irregularites in image data. The image on the left is copied onto the canvas on the right, and overlayed with plotted points that are detected "Edges" in the picture.
You may notice that the carpet brings up some "edges", these can be avoided by improving the edge-detection algorithm, making it check more pixels than just the immediate ones.

Check out the source code for an in-depth look at what's going on. The default threshold is 30.
